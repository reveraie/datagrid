# Comfort variant

The ```DataGrid``` component offers built-in support for both compact and comfortable spacing between columns. With no external styling library dependencies, all visual customization is handled within the DataGrid.css file. This allows for easy customization of the grid's appearance by simply importing and modifying your own version of the CSS file.

```tsx
    <DataGrid
        className="dg-comfort"
        columns={columns}
        rows={rows}
    />
```
